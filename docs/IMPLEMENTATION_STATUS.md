# 実装状況 Phase 7

## 実装済み

- 現在地周辺の停留所検索
- 同名停留所の統合
- 複数のりばの折りたたみ表示
- 停留所名・系統番号・行き先の検索サジェスト
- 同じのりばを使用する複数系統の統合運行画面
- 複数系統をまたいだ接近順表示
- 系統別の停留所上の車両位置表示
- 車両選択後の後続停留所への推定到着
- 複数系統を統合した直近発車予定
- 系統・行き先別の本日時刻表
- お気に入りと最近使った系統
- PWA、オフライン静的データ参照
- GTFS-RT取得のタイムアウト、再試行、フォールバック

## 現在の推定方法

公開されるVehiclePositionと静的GTFS時刻表をtrip_idで結合し、車両の現在位置における予定時刻との差を後続停留所へ加算します。公式の到着予測値ではなく、本アプリによる概算です。

## 制限

- VehiclePositionに対応するtrip_idが静的GTFS内にない車両は表示できません。
- 迂回、運休、急な運行変更は静的GTFSのみでは反映できません。
- 同じ系統・行き先でも便ごとに経路が異なる場合、停留所位置レーンは接近中の最も近い車両の便を代表として表示します。
- リアルタイム配信がCORSで直接取得できない場合はCloudflare Workerが必要です。
