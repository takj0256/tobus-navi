## 回帰対象

```text
石島 → 扇橋一丁目：4分
扇橋一丁目 → 猿江一丁目：3分
```

石島停車中から猿江一丁目までを約7分として計算することを自動テストで確認しています。

## 制限

- 停留所間は直線としてGPSを投影するため、大きく迂回する道路では誤判定する可能性がある
- `shapes.txt` による道路形状への投影は未実装
- GTFS-RT自体に含まれるGPS誤差や配信遅延は完全には除去できない
- 実道路の信号・渋滞は静的GTFS時刻表からの推定となる

次の精度改善候補は、`shapes.txt` の経路形状を変換データへ含め、直線ではなく実際の路線形状へGPSを投影することです。
