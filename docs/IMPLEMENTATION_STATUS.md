# 実装状況 Phase 8

## 実装済み

- 同名停留所を親停留所単位で統合
- 同一のりばを使う複数系統を統合表示
- 静的GTFS-JP時刻表
- GTFS-RT VehiclePositionのブラウザ内解析
- 配信時刻から現在時刻までの遅延補正
- 停留所間の予定所要時間による位置補間
- 最大30秒の先読み補正
- 短区間での先読み上限制御
- 停車中の補正停止
- 接近中の進行率制約
- GPS位置からの区間進行率推定
- 連続観測からの進行速度推定
- 停留所列上の区間内マーカー表示
- 到着時刻の範囲表示
- 後続停留所への推定到着一覧
- 10秒ごとの通常更新
- タイムアウト・フォールバック・再試行
- Cloudflare Worker中継

## 推定の前提

GTFS-RT VehiclePositionは公式の到着予測（TripUpdate）ではありません。本実装はGPS、配信時刻、車両状態、静的時刻表を組み合わせた独自推定です。

## 残る制約

- 道路形状は停留所間の直線距離比で近似
- 渋滞、信号、乗降時間の突発変動は完全には反映できない
- 位置更新が5分以上古い車両は非表示
- 位置情報がない車両は状態値から概算
- 端末を閉じている間の観測履歴は保持しない
- TripUpdateが公開された場合は、そちらを優先する余地がある
